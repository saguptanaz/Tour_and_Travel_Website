# 🌍 Tour and Travel Management System

## 📌 Overview
The Tour and Travel Management System is a web-based application developed to manage travel services like buses, trains, flights, and holiday packages. It provides both user and admin functionalities including booking, login, profile management, and service handling.

This project is built using HTML, CSS, and Java Servlet for backend processing.

---

## 🚀 Features

### 👤 User Features
- 🔐 User Registration & Login
- 🚌 Bus Booking
- ✈️ Flight Booking
- 🚆 Train Booking
- 🏝️ Holiday Packages
- 📖 View My Bookings
- 💳 Payment Page
- 👤 Profile Management

---

### 🛠️ Admin Features
- Admin Dashboard
- Manage Buses, Flights, Trains, Holidays
- User Management
- Booking Management

---

## ▶️ How to Run the Project

1. Clone the repository:
git clone https://github.com/bantishaw1/Tour_and_Travel_Website.git

2. Open in IDE (Eclipse / VS Code)

3. Configure Apache Tomcat server

4. Run the project on server

5. Open in browser:
http://localhost:8080/
